Estadistica

Mitjana, mediana, moda i representacio de dades

1 Conceptes basics

Poblacio: Conjunt total d'elements que volem estudiar.

Mostra: Subconjunt representatiu de la poblacio.

Variable: Caracteristica que mesurem (quantitativa o qualitativa).

Frequencia absoluta: Nombre de vegades que apareix un valor.

Frequencia relativa: Frequencia absoluta dividida pel total.

2 Mesures de tendencia central

Mitjana aritmetica: Suma de tots els valors dividida pel nombre total.

Mitjana = (x1 + x2 + ... + xn) / n

Mediana: Valor central quan les dades estan ordenades. Si n es parell, es la mitjana dels dos valors centrals.

Moda: Valor que mes es repeteix.

Exemple: Dades = {2, 3, 3, 5, 7} → Mitjana = 4, Mediana = 3, Moda = 3

3 Mesures de dispersio

Rang: Diferencia entre el valor maxim i el minim.

Rang = Valor maxim − Valor minim
Exemple: Dades = {2, 3, 3, 5, 7} → Rang = 7 − 2 = 5
