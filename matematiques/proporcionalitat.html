Proporcionalitat

Raons, proporcions, regla de tres i percentatges

1 Rao i proporcio

Una rao es la comparacio entre dues quantitats. Una proporcio es la igualtat de dues raons.

a/b = c/d → a × d = b × c (producte de extrems = producte de mitjos)

2 Regla de tres directa

Quan dues magnituds son directament proporcionals: si una augmenta, l'altra tambe augmenta.

a → b | c → x | x = (c × b) / a
Exemple: Si 3 kg costen 12 EUR, quant costen 5 kg? x = (5 × 12) / 3 = 20 EUR

3 Regla de tres inversa

Quan dues magnituds son inversament proporcionals: si una augmenta, l'altra disminueix.

a → b | c → x | x = (a × b) / c
Exemple: 4 obrers triguen 6 dies. Quant triguen 8 obrers? x = (4 × 6) / 8 = 3 dies

4 Percentatges

El percentatge es una fraccio amb denominador 100. Indica quantes parts de cada 100.

x% de N = (x × N) / 100
Exemple: 25% de 80 = (25 × 80) / 100 = 20

Descompte: Preu final = Preu − (Percentatge × Preu / 100)
