Operacions Basiques

Sumes, restes, multiplicacions i divisions

+ Suma

La suma (o addicio) es l'operacio que combina dos o mes nombres per obtenir un total.

a + b = c
Exemple: 25 + 17 = 42

Propietats: commutativa (a+b = b+a), associativa ((a+b)+c = a+(b+c)), element neutre (a+0 = a).

− Resta

La resta (o substraccio) es l'operacio inversa de la suma.

a − b = c (on a es el minuend, b el subtrahend, c la diferencia)
Exemple: 83 − 45 = 38

La resta NO es commutativa: a − b ≠ b − a.

× Multiplicacio

La multiplicacio es una suma repetida d'un nombre tantes vegades com indica l'altre.

a × b = c
Exemple: 12 × 7 = 84

Propietats: commutativa (a×b = b×a), associativa, distributiva respecte la suma, element neutre (a×1 = a), element absorbent (a×0 = 0).

÷ Divisio

La divisio es l'operacio inversa de la multiplicacio. Reparteix un nombre en parts iguals.

a ÷ b = c (on a es el dividend, b el divisor, c el quocient)
Exemple: 96 ÷ 8 = 12

Divisio entera: D = d × q + r (dividend = divisor × quocient + residu). No es pot dividir per 0.
